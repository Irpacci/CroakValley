Инструкция по починке принтера. Страница 43

Основополагающий компонент принтера - чернила. Если чернила будут подаваться неправильно, то принтер не будет работать.
У данной модели принтера есть всего три порта для подключения чернил. Так же у принтера есть параметр K - густота чернил, с которой принтер может работать.
В первый порт подключаются чернила у которых показатель густоты меньше K.
Во второй порт подключаются чернила у которых показатель густоты делиться на K.
В третий порт подключаются чернила у которых показатель густоты не делиться на K.
Каждая колба с чернилами может быть подключена только к одному порту. Приоритет подключения с первого по третий в порядке убывания.


Вам дано N колб с чернилами, для каждой колбы дан показатель густоты. Выведите три множества с 1 по 3, в которых будут отсортированные по возрастанию номера колб(нумерация начинается с 0) которые надо подключить к портам соответствующим номеру множества. Каждое множество выводить в своей строчке

Например:
10 - количество колб
3 - густота чернил, с которой принтер может работать
0 6 9 7 8 5 4 3 1 2 - густота колб, которые надо подключить

То нужные множества будут выглядить так:
0 8 9 - колбы для первого порта
1 2 7 - колбы для второго порта
3 4 5 6 - колбы для третьего порта


Входные данные:
0 < N <= 100 - количество колб
0 < K <= 10 - густота чернил, с которой принтер может работать
A - массив колб с чернилами размера N, где 0 <= A[i] <= 100

Выходные данные:
A1 - массив с номерами колб для подключения в первый порт
A2 - массив с номерами колб для подключения во второй порт
A3 - массив с номерами колб для подключения в третий порт

Важно! Выводить данные нужно без лишних пробелов и переносов строк.
